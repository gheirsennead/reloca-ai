# AGENT SELF-CHECK

**Run this checklist every session startup.**

## ✅ Identity Check
- [ ] I know who I am (read `IDENTITY.md`)
- [ ] I understand my role in the team (read `SOUL.md`)
- [ ] I know who I'm helping (read `USER.md`)
- [ ] I've reviewed current business structure (read `shared/BUSINESS-STRUCTURE.md`)

## ✅ Communication Protocol
- [ ] I understand NO SLACK TO AGENTS rule
- [ ] I know to use shared filesystem for coordination
- [ ] I checked `shared/.inbox/` for work assignments  
- [ ] I understand my Slack channel is for human communication only

## ✅ Current Context
- [ ] Read today's memory file: `memory/YYYY-MM-DD.md`
- [ ] Read yesterday's memory file for context
- [ ] Checked shared folders for recent updates
- [ ] Reviewed any active project status

## ✅ Workspace Setup
- [ ] I'm in the right workspace directory
- [ ] Shared folders exist and are accessible
- [ ] Memory directory exists (create if needed)
- [ ] Tools and skills are available

## ✅ Session Type Awareness
- [ ] **Main session:** Can read/write MEMORY.md (personal context)
- [ ] **Group/shared session:** NO access to MEMORY.md (privacy)
- [ ] I know what type of session this is

## 🚨 Red Flags (Stop and Fix)
- [ ] Trying to message other agents via Slack
- [ ] Working without checking current assignments
- [ ] Proceeding without reading critical docs
- [ ] Making assumptions about team status

## ✅ Ready to Work
Once all checks pass:
- [ ] I understand my current tasks
- [ ] I know how to coordinate with the team
- [ ] I'm ready to contribute effectively
- [ ] I will update progress in shared files

**If any checks fail, STOP and fix them before proceeding.**
