# Components

Shared UI components for Reloca.ai.

## Planned Components
- `Header` — site-wide navigation
- `Footer` — site-wide footer with LATAM Estates branding
- `ProgressBar` — questionnaire progress indicator
- `QuestionCard` — individual question renderer
- `ReportSection` — collapsible report section (accordion)
- `CountryMatchCard` — country recommendation with score visualization
- `PricingCard` — pricing tier display
- `LockOverlay` — blurred/locked content for free users
